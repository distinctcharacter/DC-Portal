## PDF Correction Queue

The following resources are visually usable for portal prototyping but should not be treated as final publication files. The uploaded sources are PDFs, so text correction requires an editable DOCX, Canva source, Google Doc, or a controlled rebuild.

### NCS-1 Companion Materials

Required corrections:

- Replace outdated workbook language where the resource now refers to the portal protocol experience.
- Separate client-facing `participant` language from professional `practitioner` language.
- Apply the same evidence-calibration standard used for SBP, IOS-1, and MES-1.
- Calibrate statements that present Polyvagal Theory as settled autonomic science. Retain it as an influential clinical model with scientific limitations and ongoing debate.
- Calibrate statements that present traumatic experience as literally stored in the body. Use language about physiological responses, conditioned associations, somatic cues, and body-based activation.
- Review the cold-exposure reference. If retained, state that it is optional, not a required regulation practice, and not appropriate for every health profile.
- Preserve the low-capacity adaptations, crisis boundary, medical-care boundary, licensed-provider guidance, and prerequisite sequence.

Portal implementation note:

- The portal should present the NCS tools as guided flows, selectable cards, and a searchable script library. Do not reproduce the printable document as a stack of tables.

### Biological Infrastructure Companion

Required corrections:

- Replace `diagnostic tool` with `structured self-observation tool` or `pattern-recognition tool`.
- Replace `Advanced Diagnostic Journal Prompts` with `Advanced Pattern-Recognition Journal Prompts`.
- Replace the standalone step label `Diagnosis` with `Pattern Review`.
- Keep the existing medical disclaimer.
- Add evidence-calibration language around Polyvagal Theory and body-based trauma approaches.
- Decide whether `Practitioner Notes and Framework Context` belongs in the public resource, a clearly labeled professional appendix, or the gated Practitioner Layer.

Visual note:

- The current purple-and-gold design is substantially aligned with the portal brand direction.

### 12 Dimensions of Wellness

Required redesign:

- Retain the concise content structure.
- Rebuild the printable version using the Distinct Character palette: dark purple, gold, slate grey, black, and white.
- Increase readable type size for print and mobile accessibility.
- Create a portal-native responsive version rather than using the one-page chart as the only delivery format.

## System-Level Decisions Required Before Publication

### Copyright Standard

The current files use both 2025 and 2026 copyright years. Choose one controlled copyright convention before publication. A possible standard is:

`Copyright © 2025-2026 Distinct Character. All Rights Reserved.`

Confirm the preferred legal wording before it is applied globally.

### Legal Footer

Confirm the exact approved footer order and capitalization for:

`Distinct Character`

`The Sovereign Bureau`

`A. Solenne Institute`

`A division of Granite Field Holdings Ltd. Co.`

### Named Diagnostic Framework Terms

Several framework-specific names use the word `Diagnostic`, including:

- `Fawn Diagnostic`
- `Diagnostic Verification Protocol`
- `Somatic Dysregulation Index`

These may remain as protected framework terms if they are consistently paired with clear educational-scope language. Decide whether to retain the names or rename them before the final branded standardization pass.

## Alignment Conclusion

SBP, IOS-1, and MES-1 are aligned enough to serve as controlled sources for the next implementation stage.

NCS-1, Biological Infrastructure, and 12 Dimensions of Wellness may be used as controlled references for portal structure, but they remain in the correction queue before final publication.
