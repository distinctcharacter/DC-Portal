# Distinct Character Portal
# Resource Production Roadmap v0.1

## Production Objective

Create the approved resource library before production deployment. Existing resources may continue to enter the intake process while new resources are authored, extracted, standardized, and reviewed.

## Writing Standard

- Use the Distinct Character framework and relevant psychological facts.
- Do not use filler, generic coaching language, wellness clichés, unsupported claims, or em dashes.
- Make each resource educational and operationally useful.
- Design each resource to stand alone where appropriate or clearly identify the protocol it accompanies.
- Distinguish psychoeducation from diagnosis, medical advice, and therapeutic treatment.
- Require qualified-professional review where clinical, legal, or financial scope requires it.
- Use consistent asset IDs, version labels, copyright structure, section hierarchy, and portal placement.
- Apply the universal Distinct Character cover standard to every protocol companion and resource-library document before publication.

## Intake Rule

The creator may upload existing resources at any time during this phase. Each upload will be:

1. Audited for content and formatting.
2. Assigned or matched to a permanent asset ID.
3. Classified as source material, standalone resource, protocol companion, portal-native tool, practitioner addendum, commercial resource, or duplicate.
4. Standardized or merged into the appropriate approved asset.

## Review Gates

Every completed resource must pass:

1. Framework accuracy review.
2. Psychological and clinical-adjacent language review.
3. Standalone usefulness review.
4. Portal placement and access-control review.
5. Visual formatting review.
6. Version and copyright review.
